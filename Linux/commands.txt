   Ctrl+l- для очистки терминала

   Ctrl+Shift+C= копировать в терминале

   Ctrl+Shift+V- вставить в терминале

   chtrl+Alt+T - команда открытие терминала 

    ~ - символ домашней директории (папки)

    / - символ корневой директории (папки)

   . . - обозначает преведущую дтректорию

   . - обозначает текущую директорию

 1. ls (list) - > выводить сптсок всех директории и файлов в текущей директории (рабочая директория)
    ls -a -> выводить список скрытых так и обычных файлов и директории
    ls -l -> выводить список директории и файлов с подробной информацией
    ls -la -> выводить список директории и файлов с подробной информацией скрытых так и обычных
 
 2. cd (change direktory) - > меняет текущую директорию на указанную
    cd название директории
 
 3.  mkdir(make direktory) -> создаёт указанную директорию
    mkdir <директория>
  
 4.  touch -> создаёт указанный файл
   touch <name_of_file>_<extension(расширение(txt. docs))>
   
 5. pwd(print working direktory) => выводить путь до текущей директории
   
 6.rmdir (remowe direktory) - удаляет указаный директории  (не может удалить не пустую директории)
  
 7.  rm => удаляет файлы
   rm <name_of_file>
   
 8.  rm -rf -> принудительное удаление файла либо директории
 
 9.  mv (move) -> переименование файла и перемещение файла или директории зависимости от указанного второго аргумента
   mv (файл_директория) (новое_имя_--дтректории)
   
 10. cat -> выводить содержимое указанного файла
 
 11. nana -> открывает графический интерфейс в терминале для редактирование файла
 *
     nana <name_of_file>
     
 12. echo ->  команда для быстрой записи данных в файл через терминал
 
     echo  <текст для записи >    название файла
     
 13. cp (copy) - команда для копирование файла в нужную директории
     cp (file_to_copy)  <parth>
